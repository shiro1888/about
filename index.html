§ I

五个步骤，从注册到接入

01
subscription

购买订阅

在主页选择适合你需求的订阅套餐 —— 新用户体验、轻量、中度或重度。每个套餐都包含特定的额度与有效期，按需选择，按需付费。

完成支付后，套餐立即激活，开始消费配额。

FIG. 01 / Plans
02
authorization

创建 API 密钥

进入控制台，点击 API 密钥 菜单 → 创建新密钥。给它起一个清晰可辨的名字，例如 codex-mac。

系统会生成一串以 sk- 开头的密钥，请立即复制保存 —— 这是你访问 AI 模型的钥匙。

妥善保管你的密钥，切勿分享或上传到公开仓库。
FIG. 02 / Key
03
playground

在游乐场试运行

走进 游乐场（Playground）。在这里你可以选择不同的模型，输入提示词，立刻看到响应 —— 不需要写任何代码。

这是验证密钥与额度的最快方式，也是了解每个模型脾性的好地方。

FIG. 03 / Test
04
integration

接入 Codex 与 Claude Code

在 API 密钥页面，找到针对你客户端的接入指引。我们以 Windows 端为例，借助 CC Switch 这款轻量切换工具完成配置。

Windows 客户端工具 · 推荐
CC Switch v3.14.1
在多个 AI 客户端账号之间一键切换的桌面工具，安装后会自动注册路径。
下载 MSI 安装包
Windows · MSI · 64-bit
务必下载 MSI 安装版，不要选便携版 —— 否则注册表里不会写入 CC Switch 的路径，无法被识别。

安装完成后，回到 API 密钥页面，按提示导入配置即可。

FIG. 04 / Switch
05
launch

重启即用

完成配置后，重启 Codex（或 Claude Code）一次，让它重新加载新的密钥与端点。

到此 —— 一切就绪。
