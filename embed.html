使用指南：五步上手 AI 网关

从购买订阅、创建 API 密钥，到游乐场测试和接入 Codex / Claude Code。此页面已按 NewAPI 关于页嵌入场景适配。

Step 01

购买订阅

在主页选择适合你需求的订阅套餐，新用户体验、轻量、中度或重度都可以按需选择。

完成支付后，套餐立即激活，开始消费配额。

Step 02

创建 API 密钥

进入控制台，点击 API 密钥 菜单，创建一个自己的密钥。

给密钥起一个清晰可辨的名字，例如 codex-mac，生成后立即复制保存。

密钥只展示一次，请不要分享或上传到公开仓库。
Step 03

游乐场测试

进入游乐场，选择分组与模型，输入一句测试提示词，确认密钥、额度和模型可正常响应。

Step 04

接入 Codex 与 Claude Code

在 API 密钥页面打开导入配置，以 Windows 端为例，借助 CC Switch 完成客户端配置。

Windows 客户端工具
CC Switch v3.14.1
下载 MSI 安装包
Windows · MSI · 64-bit
务必下载 MSI 安装版，不要选便携版，否则注册表里不会写入 CC Switch 路径。
Step 05

重启即用

配置写入后，重启 Codex 或 Claude Code，让客户端重新加载新的密钥与端点。

到这里，一切就绪。

开始 创造

你的 AI 网关已就位。现在，是写代码的时候了。
