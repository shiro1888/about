# About Page

ikun api 使用指南的纯静态 HTML，温暖编辑杂志风格。

## 使用

直接在浏览器访问：

https://cdn.jsdelivr.net/gh/shiro1888/about@main/about.html

或：

https://shiro1888.github.io/about/

## 文件

- `index.html` — 主页，GitHub Pages 默认入口
- `about.html` — 同样内容，便于 jsDelivr CDN 引用
- `assets/tutorial/` — 从教程文档整理出的步骤截图

## 风格

- Editorial Magazine（编辑杂志风）
- 衬线 Fraunces + 无衬线 Geist + 等宽 JetBrains Mono
- 暗色模式自动切换
